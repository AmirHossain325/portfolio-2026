## 🎬 How to Use

### 1. Quick Test
```bash
# Navigate to portfolio folder
cd a:\portfolio

# Start a local server
python -m http.server 8000

# Open browser to
http://localhost:8000
```

### 2. Customize Your Info
Edit these files with your details:
- `source/js/sections/about.js` - Your bio
- `source/js/sections/skills.js` - Your skills
- `source/js/sections/projects.js` - Your projects
- `source/js/sections/certifications.js` - Your certs
- `source/js/sections/contacts.js` - Your contact info

### 3. Adjust Colors (Optional)
Open `source/css/style.css` and modify:
```css
:root {
    --primary-color: #00ff88;     /* Green */
    --secondary-color: #0099ff;   /* Blue */
    --accent-color: #ff006e;      /* Pink */
    --neon-cyan: #00ffff;         /* Cyan */
}
```

### 4. Optimize Performance (If Needed)
In `source/js/main.js`:
```javascript
// Line 109 - Reduce particles
const particlesCount = 1500; // Instead of 3000

// Line 171 - Reduce network lines  
const lineCount = 40; // Instead of 80
```
